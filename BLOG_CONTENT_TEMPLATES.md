# Blog Content Templates (SEO Programatico)

Este arquivo traz templates prontos para publicacao em WordPress.
Use os placeholders entre chaves para gerar variacoes em escala com qualidade.

- URL slug: {{slug_curto_sem_stopwords}}
- Idioma e mercado: {{idioma_mercado}}
- CTA principal: {{cta_principal}}
- Link interno 1: {{link_interno_1}}
- Link interno 2: {{link_interno_2}}
- Link interno 3: {{link_interno_3}}
- Link externo de referencia: {{link_externo_confiavel}}

Checklist minimo por artigo:
- H1 unico com keyword principal
- Meta title e meta description unicos
- Introducao clara e objetiva
- FAQ com 3 a 5 perguntas reais
- 3 links internos e 1 externo
- Conclusao com CTA

## Modelo de QA Final (antes de publicar)

- Title unico e forte?
- Meta description unica e clara?
- H1 contem keyword principal?
- Conteudo resolve a intencao em menos de 60 segundos de leitura inicial?
- Ha exemplos reais?
- Ha links internos suficientes?
- FAQ responde duvidas de busca real?
- Conteudo nao esta duplicado/canibalizado?
- URL esta correta e curta?
- CTA esta alinhado ao funil?
